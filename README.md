# Project: Exploding Kitten Refactor

## Contributors

## Dependencies
- JDK 11
- JUnit 5.10
- Gradle 8.10

## Build Instructions
1. Press Gradle icon in IntelliJ.
2. Under course-project/Tasks/build/build right click to run to build.
3. Right-click on the Main file in the ui package and run.
